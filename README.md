影像鑑定期末 (Final Project for Computer Vision)本專案包含兩個主要實驗：紅色墨水區域面積估計與自由落體速度觀測。

透過影像處理技術與 ffmpeg 工具，對靜態影像進行幾何校正與面積計算，並對動態影片進行影格分析以驗證物理運動定律 。

專案目錄結構:

FreeFall.MOV: 自由落體實驗原始影片。

final.py: 影像處理與面積計算之 Python 腳本。

影像鑑定期末報告.pdf / .docx: 完整實驗流程與數據分析報告。

11477007_final.zip: 專案完整備份壓縮檔。

test1.jpg / code.png: 實驗測試圖與程式截圖。
